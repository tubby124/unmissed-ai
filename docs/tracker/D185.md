# D185 — Mode-First Onboarding

## Problem
Onboarding shows all steps regardless of mode. message_only clients go through FAQ/knowledge steps that are irrelevant to them.

## Solution
Show mode selector early (step 2). Based on mode:
- **AI Voicemail** → skip FAQ, knowledge scrape, service catalog
- **Smart Receptionist** → show FAQ + knowledge
- **Receptionist + Booking** → show everything + calendar
