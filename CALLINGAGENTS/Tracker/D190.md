# D190 — Feature Unlock CTAs

## Problem
Locked features just show nothing. No CTA, no explanation. User doesn't know what they'd get or how to unlock it.

## Solution
Locked feature click → modal:
- What this feature does (1 sentence)
- What plan it requires
- One action button: "Upgrade to unlock" or "Configure now" (if free)
- Target: inactive → active in under 30 seconds
