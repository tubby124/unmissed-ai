# D357 — Greeting confidence for new onboardings

## Problem
New agents open with "Red Swan Pizza AI Assistant, I can do X, Y, Z" — the opening feels robotic and lacks confidence. The capability signaling is good but the delivery isn't natural enough.

## Root Cause
The `GREETING_LINE` from the intelligence seed (Haiku-generated) produces a templated opening. The prompt slot `greetingSlot()` in `src/lib/prompt-slots.ts` uses it verbatim.

## Fix Options
1. Improve the Haiku prompt that generates GREETING_LINE to produce more conversational, confident openers
2. Add voice coaching in the prompt slot: "Deliver your greeting with confidence — don't sound like a recording"
3. Provide 3 greeting variations and let the agent rotate

## Files
- `src/app/api/onboard/generate-agent-intelligence/route.ts` — Haiku prompt
- `src/lib/prompt-slots.ts` — greetingSlot()
