# D363 — ShareNumberCard Lost Functionality

## Problem
The notification grid redesign replaced `ShareNumberCard` with a simple grid nudge item that links to settings. The original card had:
- Copy-to-clipboard button with formatted number display
- Per-carrier forwarding codes (Rogers/Fido, Bell/Virgin, Telus/Koodo)
- Expandable carrier code section
- Session-dismissible

The grid item only shows "Share your number" + formatted phone + link to settings.

## Fix (2026-04-01)
Changed the `share-number` grid item from a settings link to an inline expand toggle:
- Click "View" → expands a panel below the grid with:
  - Formatted number display + **Copy** button (clipboard)
  - All 3 carrier forwarding codes (Rogers/Fido/Chatr, Bell/Virgin/Lucky, Telus/Koodo/Public Mobile)
  - Forward + Cancel codes with the client's actual Twilio number
- Click "Close" → collapses
- No page navigation required

- `src/components/dashboard/home/UnifiedHomeSection.tsx` — grid item → onClick toggle + expanded panel
