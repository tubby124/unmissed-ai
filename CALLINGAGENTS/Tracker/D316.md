# D316 — Fix hardcoded caller reason examples in step3-capabilities

## Bug
`src/app/onboard/steps/step3-capabilities.tsx:471-473` — "Why do people call you" placeholders hardcoded:
```
"Need a windshield replaced"
"Rock chip — want a price"
"Insurance claim question"
```
Shows for ALL niches. Real estate user sees windshield examples.

## Fix
Create a `NICHE_CALLER_REASON_EXAMPLES` map keyed by niche. Use selected niche for placeholder text. Similar pattern to `NICHE_SERVICE_SUGGESTIONS` and the `NICHE_PH` map in `QuickConfigStrip.tsx:171-178`.
