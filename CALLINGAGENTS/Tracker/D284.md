# D284 — Self-Improving Agent Loop

## Vision
As calls happen, the agent gets smarter automatically:
1. Knowledge gaps surface (caller asked about X, agent didn't know) → D252
2. AI suggests prompt improvements based on patterns → D257
3. Owner confirms or tweaks with one click → D251
4. Agent updates live → call quality improves next call

The flywheel: more calls → more data → better agent → more value → more calls.

## Components
- D252: Knowledge gap → one-click fix CTA
- D257: AI-assisted prompt suggestions feed (Haiku generates improvements)
- D251: Per-section prompt editor (owner can self-serve fix)
- D250: ROI card shows improvement over time

## Status
Open — architectural item. Individual components tracked separately.
