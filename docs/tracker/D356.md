# D356 — Telegram notification preview on Overview

## User feedback
"We had a place where it showed like notifications like how it looks when Telegram sends message out but we are not seeing it in Overview. Don't you think it would be valuable to see what the Telegram output looks like so kitchen can see it, confirm the order and book it for them?"

## Problem
After a call, the Telegram notification contains the structured summary (caller name, reason, lead temperature, etc.). But:
1. Test calls don't trigger Telegram notifications
2. Even for real calls, the owner has to check Telegram separately
3. There's no in-dashboard preview of what the notification looks like

## Proposed
Add a "Last Notification" or "Notification Preview" card on Overview:
- Shows a mock/real Telegram message format for the most recent call
- For test calls: shows "This is what your Telegram notification would look like"
- Includes: caller name, reason, lead score, summary, action items
- Helps the owner understand the value of setting up Telegram before they commit

For restaurant niche specifically: the Telegram message could include order details formatted for kitchen use — item list, pickup/delivery, time, name.

## Files
- Overview page — add notification preview card
- Reuse notification formatting from `lib/notifications.ts` or Telegram alert code
