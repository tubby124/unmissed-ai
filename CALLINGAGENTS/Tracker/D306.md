# D306 — Agent Brain Empty States

Every card on the Overview needs a "getting started" CTA for trial users and users with incomplete setup.

## What This Covers
- Trial user sees 0 calls → "No calls yet — forward your number to get started"
- Knowledge Base empty → "Add your website or business facts"
- Unanswered Questions empty → "Your agent will track questions callers ask"
- Bookings empty → "Connect Google Calendar"
- Google Listing not connected → "+ Connect Google listing"
- Business Facts empty → "Add business facts so your agent can answer questions"
- Agent Setup Progress ring (25%, 50%, 75%, 100%) with next-step CTA

## Key Principle
Trial users see the SAME layout as paid users — just with helpful empty states and CTAs instead of data. No separate "trial dashboard" vs "paid dashboard" — one layout, data fills in.

## Dependencies
- D278 (Agent Brain page) must define the layout first
- All existing empty state components can be reused

## Files
- `src/components/dashboard/home/UnifiedHomeSection.tsx` — current empty states
- `src/components/dashboard/home/TrialActiveSection.tsx` — trial-specific
- `src/components/dashboard/home/SoftTestGateCard.tsx` — setup progress

## Acceptance
- [ ] Every tier in the matrix has a trial-user empty state
- [ ] Agent Setup Progress ring visible for incomplete users
- [ ] Each empty state has a specific CTA that links to the right setup step
