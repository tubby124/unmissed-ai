# D364 — color-mix() Browser Compatibility

## Problem
The nudge grid uses `color-mix(in srgb, ${n.color} 12%, transparent)` for icon backgrounds and dismiss button backgrounds. This CSS function requires:
- Chrome 111+ (Mar 2023)
- Safari 16.4+ (Mar 2023)
- Firefox 113+ (May 2023)

Older browsers will silently ignore the property, leaving icon backgrounds transparent and dismiss buttons invisible.

## Fix
Replace `color-mix()` with hardcoded rgba values per color variant, or use a Tailwind utility class instead.
