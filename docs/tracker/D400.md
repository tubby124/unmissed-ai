# D400 — Add hvac + plumbing to PRODUCTION_NICHE_KEYS

**Status:** ✅ Done
**Date:** 2026-04-04

## What
Added `hvac` and `plumbing` to `PRODUCTION_NICHE_KEYS` in `niche-defaults.ts`.
Updated `NICHE_KEYWORD_MAP` so `hvac`/`heating`/`cooling`/`furnace`/`air-condition` route to the `hvac` template, and `plumbing`/`plumber`/`drain`/`pipe`/`sewer` route to the `plumbing` template — no longer fall through to `auto_glass`.
Also added missing `LINGUISTIC_ANCHORS` to the plumbing NICHE_DEFAULTS block.

## Why
HVAC and plumbing both had full NICHE_DEFAULTS entries but were silently resolving to `auto_glass` via `resolveProductionNiche()`. Agents for these niches were using the auto glass template instead of the dedicated one.
