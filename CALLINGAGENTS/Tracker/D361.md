# D361 — Call-me card: no "call again" after success

## Problem
Once `callMeSuccess` is set to `true`, the call-me card shows the success message permanently until page refresh. User can't trigger another call without reloading.

## Fix
Add a "Call again" link or auto-reset the success state after ~10 seconds so the user can re-enter a number and call again.
