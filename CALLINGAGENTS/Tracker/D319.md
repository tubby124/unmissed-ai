# D319 — Simplify voice picker to Male/Female first

## Problem
Step 2 shows all available voices at once via `GenderVoicePicker`. Too many options for onboarding — decision fatigue.

## Fix
Replace the full voice grid with a simple 2-option toggle:
- **Male voice** → default male (Mark/Blake depending on niche)
- **Female voice** → default female (Jacqueline)
- Small link: "More voices available in your dashboard"

The niche already determines the best default voice via `NICHE_VOICE_MAP` in `niche-config.ts`. Onboarding just needs Male vs Female as the primary choice.

## Files
- `src/app/onboard/steps/step2-voice-preview.tsx`
- `src/components/onboard/GenderVoicePicker.tsx`
- `src/lib/niche-config.ts` — `NICHE_VOICE_MAP` already maps niche → voice

## Size
Medium — UI redesign of voice picker for onboarding context.
