# D432 — 4 idle Twilio numbers accruing $4.60/month

**Status:** open
**Priority:** P3
**Filed:** 2026-04-06

## Problem
4 Twilio numbers are provisioned but not assigned to any active client, accruing ~$1.15/month each (~$55/year total).

## Root Cause
Numbers were provisioned during testing/demos and never released.

## Fix
Go to Twilio console → Phone Numbers → list all numbers → identify any not in clients.twilio_number → release them.

## Notes
Low urgency but easy money. Run a SQL query first: `SELECT twilio_number FROM clients WHERE status = 'active'` then compare against Twilio console list.
