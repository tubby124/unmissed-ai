# D351 — Overview knowledge card counts from ALL sources

## Problem
KnowledgeInlineTile showed `1` next to every source instead of actual chunk counts. The code had `count: isActive ? 1 : 0` — binary presence check, not real count.

## Fix
1. `src/app/api/dashboard/home/route.ts` — added `sourceCounts` map (groups approvedChunks by source, counts each)
2. `src/components/dashboard/ClientHome.tsx` — added `source_counts?: Record<string, number>` to knowledge type
3. `src/components/dashboard/home/KnowledgeInlineTile.tsx` — uses `def.sources.reduce((sum, s) => sum + (sc[s] ?? 0), 0)` for real per-source counts

Also fixed: "Pending" badge now shows "15 pages imported" in green when chunks are approved even if `website_scrape_status='extracted'`.
