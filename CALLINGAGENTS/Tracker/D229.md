# D229 — "Call Back Now" Button on Call Rows

## Problem
Owner sees HOT/WARM lead in call log but has to manually dial. High friction = leads go cold.

## Solution
HOT/WARM call row shows "Call back →" button.
Click → POST /api/dashboard/test-call with caller's toPhone.
Owner's phone rings → connected via agent bridge.

## Why Easy
Route already supports toPhone param. Just needs a button + lead-tier gate in the UI.
