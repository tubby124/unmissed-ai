# D411 — Calendar Auth for PM (Showing Scheduling)

## Problem
PM niche has a natural use case for showing scheduling — prospective tenants book viewings via Jade. The existing calendar/booking system works for other niches but hasn't been wired for PM: the "service type" concept maps awkwardly to "property showing."

## Fix
- Enable `booking_enabled` flag for PM clients
- Map showing requests to calendar slots (service type = "Showing", duration configurable)
- Add "Available units" context so Jade can mention which units are available for showings
- Ensure `[SHOWING REQUEST]` in D415 Telegram format triggers on actual booked showings, not just mentions

## Files
- `src/lib/prompt-config/pm-ops.ts` — wire showing booking context
- `src/app/onboard/steps/niches/property-management.tsx` — add "enable showing scheduling" toggle
- `src/lib/prompt-slots.ts` — PM showing-specific booking instructions
