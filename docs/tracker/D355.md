# D355 — Quick-view modal for knowledge sources on Overview

## User feedback
"Should be like a popup modal that links to the actual place but quick view style to add or remove items and if they wanted to view the whole thing they can go there and do it but it should be everything possible in overview."

## Current state
- Overview Knowledge Base card shows source tabs (Website, AI Compiler, etc.) with counts
- Clicking a source: expands to show "No approved chunks yet" or "Add content →" — but doesn't show the actual items that ARE there
- User uploaded 12 menu items but can't see them from Overview

## Proposed
Clicking a source on Overview opens a **quick-view modal**:
- Shows list of chunks from that source (scrollable)
- Each chunk: content preview, category badge, edit/delete actions
- "Add more" button at top
- "View all in Knowledge →" link to full Knowledge page
- Close modal to stay on Overview

This lets the owner manage knowledge without leaving Overview. The full Knowledge page is for power users who want to do bulk operations.

## Files
- Overview page — add modal trigger to source tabs
- New: `QuickKnowledgeModal.tsx` or similar
- Reuse chunk display components from Knowledge page
