# D308 — Agent Brain Navigation Placement

Design decision: where does "Agent Brain" (the new Overview) sit in the nav? Run with D278 + D286.

## Current Nav
```
OVERVIEW | KNOWLEDGE (9+) | CALLS & LEADS | BILLING | SETTINGS | OTHER
```

## Options
1. **Rename OVERVIEW → AGENT BRAIN** — zero nav changes, just a rebrand
2. **Keep OVERVIEW, add AGENT BRAIN as sub-page** — more complex, confusing
3. **OVERVIEW stays as-is, Agent Brain is a section WITHIN overview** — least disruptive

## Recommendation
Option 1. The Overview IS the Agent Brain. Rename the tab. The content reorganization (D278) makes it worthy of the name.

## Decision Criteria
- Trial users need to land here by default
- "Agent Brain" communicates intelligence/control better than "Overview"
- No additional nav items — the current tab count is already at the limit for mobile

## Dependencies
- D278 (Agent Brain page content)
- D286 (Settings reorg — may affect what moves OUT of Settings INTO the brain)

## Acceptance
- [ ] Nav label decided and implemented
- [ ] Default landing page unchanged (still /dashboard)
- [ ] Mobile bottom tab bar updated if renamed
