# D322 — Loading orb during GBP lookup

## Problem
During GBP search and place details fetch, the only loading indicator is a tiny 4px spinner inside the search input. No full-page or section-level indicator.

## Fix
Show the same voice-powered orb (WebGL component from `VoicePoweredOrb`) used on the Launch step during:
1. Place details fetch (after clicking a prediction)
2. Optionally during initial autocomplete

Could be a smaller version of the orb with text like "Looking up your business..." shown where the confirmation card will appear.

## Files
- `src/components/onboard/PlacesAutocomplete.tsx` — loading states
- `src/app/onboard/steps/step1-gbp.tsx` — render loading state between prediction click and confirmation card

## Size
Small-medium — reuse existing orb component.
