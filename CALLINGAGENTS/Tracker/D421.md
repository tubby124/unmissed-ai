# D421 — NICHE_EXAMPLES Gaps for PM

## Problem
`NICHE_EXAMPLES` (few-shot conversation examples) for property_management are incomplete or missing. The prompt uses generic examples. PM-specific scenarios (maintenance request, showing inquiry, noise complaint, lease question, emergency escalation) need realistic examples to guide Jade's tone and handling.

## Fix
Add 5–8 PM-specific examples to `NICHE_EXAMPLES.property_management` covering:
1. Routine maintenance request → create ticket, confirm next steps
2. Emergency (no heat in winter) → P2 urgent, immediate escalation
3. Showing inquiry → collect info, route to scheduling
4. Noise complaint → take message, note unit
5. Lease/rent question → "I'll have the property manager follow up"
6. Package/delivery question → reference package policy
