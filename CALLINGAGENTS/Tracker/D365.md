# D365 — ClientHome Parent Banners Still Full-Width

## Problem
We redesigned the banners inside `UnifiedHomeSection` into a compact 3-layer system, but `ClientHome.tsx` still renders full-width banners ABOVE it:
1. `LiveCallBanner` — realtime active call banner (acceptable — ephemeral)
2. Upgrade success banner (lines 385-401) — full-width card with text + link
3. `TrialWelcomeBanner` — full-width dismissable trial welcome

A trial user can see: TrialWelcomeBanner (full-width) + toast pills + action grid + call-me card. The parent banners undo some of the vertical space savings.

## Options
1. Convert upgrade success + trial welcome to toast pills (same pattern as Layer 1)
2. Merge TrialWelcomeBanner into the nudge grid as a wider card
3. Leave as-is — these are one-time/dismissable and acceptable
