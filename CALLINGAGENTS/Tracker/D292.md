# D292 — Guided Call Forwarding Wizard

## Summary
#1 activation blocker. User completes onboarding, gets a Twilio number, has no idea what to do with it. Without call forwarding, the agent never rings. The post-activation screen currently shows "Forward your calls" as a checklist item with no instructions.

Ask carrier → show exact steps + dial codes → "Test it now" button → confirm working. We have carrier data in memory/canadian-carrier-call-forwarding.md.

## Why
Every single new client hits this wall. You can have a perfect prompt and a working agent — if they don't forward their business number to the Twilio number, it takes zero real calls. This is the last mile and it is completely undocumented in the product. #1 friction point in ALL voice agent platforms.

## What exists
- Post-activation checklist has "Forward your calls" as item #1 but no instructions
- `memory/canadian-carrier-call-forwarding.md` has dial codes per carrier
- The Twilio number is shown in the dashboard Settings

## Fix
On the activation success screen and in dashboard Settings → Numbers tab:
1. "Which carrier is your business phone on?" → Rogers / Bell / Telus / Other
2. Show exact dial sequence: `*72 [Twilio number]` (Rogers/Bell) or carrier-specific
3. "Test it now" → trigger a test call to the Twilio number → confirm ring
4. Mark forwarding confirmed in Supabase (`clients.call_forwarding_confirmed = true`)

## Phase
Independent (doesn't depend on architecture refactor)

## Resolution (2026-04-06)
`SetupView.tsx` already contains the full forwarding wizard:
- Line type picker (Landline / Mobile / VoIP)
- Carrier picker from `LANDLINE_CARRIERS` array
- Star codes shown per carrier with copy buttons
- Mobile GSM codes (`*61*number#`, `*67*number#`, `*62*number#`)
- VoIP portal navigation instructions
- Activation state machine (`forwarding_needed` → `active`)
- "Update forwarding codes" collapsible for phone/carrier changes

Reached from `/dashboard/setup` when `activationState === 'forwarding_needed'`. Shown automatically post-paid activation.
