# D189 — Unify Trial/Paid Dashboard

## Problem
Trial dashboard hides IVR, SMS toggle, Telegram, and other features. Creates confusion — trial user doesn't know what paid plan includes. Free-to-configure features (Telegram, IVR) should be accessible in trial too.

## Solution
One dashboard view for trial and paid.
- Locked features: show "upgrade to unlock" preview card, not blank
- Free-to-configure features: accessible immediately in trial
