# D186 — Mode Capability Preview in Dashboard

## Problem
Mode selector (message_only / info_hub / booking) has no live preview. User selects a mode without understanding what the agent will/won't do.

## Solution
Mode selector shows 3-panel preview as mode changes:
- "In AI Voicemail mode: agent will [X], will NOT [Y]"
- Live preview updates instantly on selection
