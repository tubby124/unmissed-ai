# D421 — PM NICHE_EXAMPLES gaps: pest reports, lease renewal, language barrier

**Status:** 🔴 Open
**Priority:** LOW-MEDIUM
**Date:** 2026-04-05

## Problem
NLM research identified 3 missing inline dialogue examples in the PM NICHE_EXAMPLES section:
1. **Pest reports** — caller says "I have cockroaches / mice." PM may have "pest_control" in servicesNotOffered, but the agent needs an example of how to handle this gracefully.
2. **Lease renewal questions** — "My lease is up in 2 months — can I renew?" Current TRIAGE_DEEP routes to callback but there's no example showing how to do this without alarming the tenant.
3. **Language barrier callers** — caller is non-English or broken English. No guidance on how Jade should respond (slow down, confirm each piece of info, never ask to call back because "we have an English line").

These are not critical but they're the edge cases that cause awkward calls and make a PM not trust the product.

## Solution
Add 3 new examples to the PM `NICHE_EXAMPLES` block in `niche-defaults.ts`:

**Example D — Pest report (excluded service):**
```
Caller: "hi there's cockroaches in my unit"
You: "oh no, that's not good — what's your name and unit number?"
Caller: [name and unit]
You: "got it [name], I've logged that for the property manager at unit [X]. Pest issues are something {{CLOSE_PERSON}} handles directly — they'll call you back to arrange next steps. talk soon." [use hangUp]
[Note: don't apologize or over-explain. Log it, route it, close confidently.]
```

**Example E — Lease renewal:**
```
Caller: "my lease is up in 2 months, I want to renew"
You: "great — I'll let {{CLOSE_PERSON}} know you're interested in renewing. what's your name and unit?"
Caller: [name and unit]
You: "perfect, I've noted that [name] at unit [X] wants to discuss renewal. {{CLOSE_PERSON}} will be in touch. talk soon." [use hangUp]
```

**Example F — Language barrier:**
```
[Caller's English is limited — short sentences, may repeat themselves]
You: slow down, confirm each piece of info one at a time.
"okay — your name?" [pause] "and your unit number?" [pause] "and what's the problem?" [pause]
Confirm back slowly: "got it — [name], unit [X], [issue]. I'm flagging that for the property manager. They'll call you back. okay?"
NEVER ask them to call a different number. NEVER say "maybe someone else can help you."
```
