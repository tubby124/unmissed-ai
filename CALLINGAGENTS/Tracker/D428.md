# D428 — Stripe test mode — every activation requires manual admin

**Status:** open
**Priority:** P0
**Filed:** 2026-04-06

## Problem
Stripe is in test mode on Railway. Every new client that signs up requires Hasan to manually go into admin and click Generate Prompt + Activate. Cannot hand off or scale until this is live.

## Root Cause
STRIPE_SECRET_KEY and STRIPE_WEBHOOK_SECRET on Railway are still test-mode keys.

## Fix
(1) Go live with Stripe account, (2) Swap Railway env vars to live keys, (3) Update webhook endpoint in Stripe dashboard to production URL. No code changes needed.

## Notes
This is the primary reason unmissed is still $0 revenue. Every client is on free manual activation.
