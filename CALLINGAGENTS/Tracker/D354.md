# D354 — Unanswered questions prominent placement under orb

## User feedback
"I think [unanswered questions] should be 1/3 column not a 1/1 column and just underneath the orb so they can easily see the unanswered questions and answer it."

## Current state
- Unanswered questions appear as a full-width bar at the bottom of Overview
- Easy to miss — below the fold, not prominent
- Shows count badge and "Answer / Skip" actions — good data, bad placement

## Proposed layout
Move unanswered questions into the right 1/3 column, directly below the test call orb:
```
[Knowledge Base 2/3]  [Test Call Orb 1/3]
                      [Unanswered Qs   ]
                      [Voice Style     ]
                      [Today's Update  ]
```

This puts the learning loop right next to the test tool — user sees a gap, answers it, re-tests immediately. Tight feedback loop.
