# D283 — All Prompt Variables Visible + Editable on Dashboard

## Problem
Users can't see what template variables are in their agent's prompt. They don't know what `{{TRIAGE_DEEP}}` contains, what `{{SERVICES_OFFERED}}` looks like, or what `{{PERSONALITY_LINE}}` says. They have no visibility into what drives their agent's behavior.

## Vision
Dashboard "Agent Brain" section shows every variable:
- AGENT_NAME: "Dave" [Edit]
- BUSINESS_NAME: "Plumber Calgary NW" [Edit]
- PERSONALITY_LINE: "Upbeat and relaxed..." [Dropdown: casual/professional/direct/empathetic]
- SERVICES_OFFERED: "Toilet repair, pipe fixing..." [Edit via service catalog]
- TRIAGE_DEEP: "When caller says X, ask Y..." [Edit via call routing card]
- HOURS_WEEKDAY: "Mon-Fri 8am-5pm" [Edit via hours card]
- CLOSE_PERSON: "Emon" [Edit]

Each variable links to the settings card that controls it.
