# D318 — Step 3 bloat — trim to mode selection only

## Resolution
Step 3 deleted entirely in cbbe261 (Phase 7 onboarding simplification). The 7-step flow collapsed to 3 steps: GBP → Plan → Launch. Agent mode is now derived from plan selection (D324). All step 3 inputs (services, FAQ, caller reasons, urgency, forwarding, IVR) moved to dashboard or step 1.

## Original Problem
Step 3 contained 10+ distinct inputs causing overwhelming UX and abandonment.
