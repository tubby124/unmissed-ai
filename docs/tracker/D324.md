# D324 — Plan = Mode (derive agent mode from plan selection)

## Resolution
Shipped in cbbe261 (Phase 7). `planToMode()` utility derives `call_handling_mode` from `selected_plan`:
- Lite → `voicemail_replacement`
- Core → `lead_capture`
- Pro → `appointment_booking`
- Trial → `lead_capture` (all features unlocked)

Step 3 (mode selector) deleted. Mode is now implicit from plan choice.
