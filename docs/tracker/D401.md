# D401 — Wire {{INSURANCE_TYPE}} into HVAC TRIAGE_DEEP

**Status:** ✅ Done
**Date:** 2026-04-04

## What
Added insurance mention handling to the HVAC `TRIAGE_DEEP` diagnostic questions block:
> "If caller mentions insurance: 'Insurance accepted: {{INSURANCE_TYPE}} — just so you know before we send someone out.'"

## Why
NLM research (notebook 8d82f254) confirmed that the one data point that makes HVAC agents pass the skeptic test is immediately confirming insurance acceptance when a caller mentions it. Without this, callers who use insurance get routed to a human callback unnecessarily.

The `{{INSURANCE_TYPE}}` slot is already populated via the `['insurance', 'INSURANCE_TYPE']` directMapping in `prompt-slots.ts`.
