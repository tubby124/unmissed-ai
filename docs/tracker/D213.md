# D213 — Per-Section Prompt Editor UI

## Problem
Backend is fully done: replacePromptSection() + PATCH /api/dashboard/settings accepts section_id + content.
But there is NO UI that renders prompt sections as editable blocks.

## Solution
UI in PromptEditorCard that renders each named section as an expandable edit block:
- OPENING, TRIAGE, CALL HANDLING MODE, INFO COLLECTION, AFTER HOURS, ESCALATION, EDGE CASES
- Each section: expand → edit → Save | Reset to default
- Save calls existing PATCH route with section_id + content

## Files
- Backend done: `src/lib/prompt-sections.ts` → replacePromptSection()
- UI needed: `src/components/dashboard/settings/PromptEditorCard.tsx`
