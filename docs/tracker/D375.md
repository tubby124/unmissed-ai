# D375 — Admin Recompose Button

## Summary
Push PERSONA sandwich (D377) and LINGUISTIC_ANCHORS (D378) to existing clients without manual Supabase edits. One-click prompt rebuild + Ultravox sync per client.

## What Was Done (2026-04-04)

### Route
`POST /api/admin/recompose-client` — body: `{ client_id }`, admin-only auth.
Calls `recomposePrompt(clientId, userId)` from `slot-regenerator.ts`.
Returns `{ ok, promptChanged, charCount }`.
Guard: no-op if prompt unchanged, 422 if old-format prompt (must migrate to slot markers first).

### UI
`AdminRecomposePanel` component rendered in the Billing tab admin-only section (alongside `AdminPromoPanel`).
Shows: Recompose button → success message with char count, or error.
