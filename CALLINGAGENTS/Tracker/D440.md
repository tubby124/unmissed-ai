# D440 — Learning Bank platform: extend call_insights into actionable feedback loop

## Problem
`call_insights` had 26 rows of useful signal (loop_rate, repeated_questions, caller_frustrated, agent_confidence) with no consumer. `knowledge_query_log` had 261 rows of unresolved gaps with no consumer. Every prompt improvement was a manual one-off; cross-niche learnings stayed locked inside individual prompts.

## Solution
Build a Learning Bank: 4 new tables (`prompt_patterns`, `prompt_lessons`, `pattern_application_log`, `call_transcripts`) + 1 view (`v_active_patterns_by_niche`). Auto-generate lessons from `call_insights` thresholds at `/completed`. Promote validated lessons into a reusable pattern library. Inject promoted patterns into `recomposePrompt()` (gated, default OFF until per-niche validated).
