# D366 — Call-Me Card Mobile Layout

## Problem
The call-me card was redesigned from stacked (label above, input below) to inline (label left, input+button right). The input is `w-36` (144px) which on a 375px screen with padding leaves very little room.

Previous layout stacked naturally on mobile. New layout may require horizontal scrolling or cause the input to be too narrow to type a phone number.

## Fix
Add a responsive breakpoint: stack on mobile (`flex-col` below `sm:`), inline on desktop.
