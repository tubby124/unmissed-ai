# D446 — Harden drift-detector tool-extractor for all 3 wire shapes

## Why

D444 was opened on a "100% of clients are missing `hangUp` from `clients.tools`" finding from the D442 Phase 1 audit. Investigation found the audit's premise was **wrong** — the drift-detector script's tool-name extractor failed on entries that didn't match its expected shape and silently dropped them, producing a symmetric miscount across DB and Ultravox sides.

Both sides had `hangUp`. They just used different wire shapes:
- DB `clients.tools[0]`: `{ "toolName": "hangUp", "parameterOverrides": { "strict": true } }` — built-in tool by name
- Ultravox `callTemplate.selectedTools`: `{ "toolId": "...", "nameOverride": "hangUp", ... }` — built-in tool by ID with name override

The script handled neither cleanly when neither `temporaryTool.modelToolName` nor a recognized wrapper was present. Result: false 100%-of-clients contract-violation alarm.

## What to fix

Update the drift-detector subagent (`/Users/owner/Downloads/CALLING AGENTs/.claude/agents/drift-detector.md`) and any inline scripts it generates so the tool-name extractor handles all three wire shapes:

1. **Built-in by name:** `{ toolName: 'hangUp', parameterOverrides? }` — name = `toolName`
2. **Built-in by ID with override:** `{ toolId: '<uuid>', nameOverride: 'hangUp', parameterOverrides? }` — name = `nameOverride`
3. **Inline custom tool:** `{ temporaryTool: { modelToolName: 'sendTextMessage', ... }, parameterOverrides? }` — name = `temporaryTool.modelToolName`

A tool entry that matches none of these is a real bug — but log it with the raw JSON, don't silently drop it.

## Why this matters beyond D444

The drift-detector is the project's authoritative read-only audit tool. Any future Phase 1-style audit will use it. Bug stays buried until another false alarm lands a ticket on the wrong target.

Pattern is general: any audit script comparing two complex JSON sources must normalize across all known wire shapes BEFORE comparing, and must surface unmatched shapes loudly rather than dropping them. Recorded as durable concept: [[unmissed-audit-instrument-verification]].
